Install and configure Openfire XMPP Server - Debian, Ubuntu

Difficulty: 2

Time: 20 minutes

This article will help you to understand the installation and configuration of Openfire XMPP Server on Debian or Ubuntu.

Extensible Messaging and Presence Protocol (XMPP) is an XML-based communication protocol for middleware. It enables the near-real-time exchange of structured yet extensible data between any two or more network entities.

Openfire enables collaboration in real time and provides a tightly secured and high performing server. You can use Open Source Apache License for Openfire and can easily set up and configure it using Java.

Openfire is a java based tool to use utility of XMPP server.

Openfire disallows chat options.

Install Openfire

  1. Update the operating system.
    sudo apt-get update
  2. To install java, run the following command:
    sudo apt-get install default-jre
  3. Download Openfire form source.
    sudo wget -O openfire.deb http://www.igniterealtime.org/downloadServlet?filename=openfire/openfire_3.8.2_all.deb
    This command initiates the download of the Openfire Debian package to openfire.deb.
  4. Install the openfire.deb package.
    sudo dpkg --install openfire.deb
  5. Start the service again after every reboot.
    sudo chkconfig openfire on
  6. Start the service.
    sudo /etc/init.d/openfire start

  7. Test Openfire

    1. Test the Openfire installation on your system, by visiting the following URL on your browser :
      http://your_IP:9090
    2. Successful installation of Openfire would show the Openfire wizard on your browser. Follow the instructions to proceed with the set up.
    3. On the first page, choose the language.
    4. On the second page, provide appropriate values for domain and associated ports.
    5. Leave the default values for the next steps.
    6. Once completed with the setup procedure, login into the admin console using the Admin account.
    7. Once logged-in, the first screen shows the server manager tab under the Server tab. The server manager provides information about server properties and environment, system properties, language and time, clustering, database, logs, email setting and security audit viewer.
    8. The server setting tab provides information about profile settings, client connections, server to server connections, external components, http bindings, update management, registration and login management, resources policies, offline messages, audit policies for messages, private data storage options, security settings, server certificates, compression settings and file transfer settings.
    9. The Media services tab provides information about Media Proxy settings.

    Conclusion

    By using all of the above steps, you will be able to install and configure openfire XMPP server on Debian and Ubuntu.
