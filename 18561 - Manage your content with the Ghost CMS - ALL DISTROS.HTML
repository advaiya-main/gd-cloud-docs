Manage Your Content with Ghost CMS - ALL DISTROS

Difficulty: 3
Time: 1.5 hrs

Ghost is a powerful blogging platform. Besides simplicity in setup, it has a powerful user interface that lets you manage your content with ease. This article provides an introduction to various content management options provided by the Ghost blogging platform.

Prerequisites

This article assumes that the host machine has a Ghost blogging platform deployed and running. For instructions, see one of the following:

  • Install and configure Ghost with NGINX - Ubuntu
  • Install and configure Ghost with NGINX - CentOS, Fedora
  • Install and configure Ghost with NGINX - Arch Linux
  • Install and configure Ghost with NGINX - Debian

Ensure that Ghost is running

  • Type the IP address or the domain name: http://192.169.170.227:2368 or http://coolexample.com in the browser.

Start a new blog with Ghost

  1. Open the blog setup page in the browser: http://192.169.170.227:2368/ghost/setup/
  2. Provide the required details in the blog setup page.
  3. After the registration is complete, the Content tab shows the sample preconfigured blog as Welcome to Ghost.

Create a new post

  1. Click on the NEW POST icon on the top left corner to create a new post.
  2. Type in or past the title for the post and the content in the provided windows. While putting the content, the preview of the post is simultaneously visible in the PREVIEW window.
  3. Access the help regarding formatting the content via the small icon on the top right corner of the Markdown window (the content editing window). For formatting the content, Ghost uses its own markup language called markdown, which uses simple markdowns for formatting content.
  4. The Markdown Help window shows all the options for additional formatting of text, like bold, emphasize, links and so on.
  5. Option for adding tags is available in the bottom left corner.
  6. Option to publish the draft, save the draft or delete the post are available in the bottom right corner of the window.
  7. Additional post settings like post URL, publish dates, author name and other meta data can be accessed using the Settings icon, next to the SAVE DRAFT button in the bottom right corner.

Additional blog settings and information

  1. The Settings tab provides additional blog settings, that includes the following:
    • General: General information about the blog, like title, description, logo, cover images, email address, number of posts per page, permanent links with dates and blog themes.
    • Users: List of active users of the blog.
    • Tags: Tags applied across all the articles in the blog.
    • Navigation: Additional navigation options besides Home.
    • Code Injection: Adds external code into the top and bottom of blog theme, without editing the theme files.
    • Labs: A testing ground for experimental features.
    • About: Information about Ghost version, environment, database, mail and other additional information.
  2. Additional help options can be viewed via the question-mark icon, next to the User Name at the top right corner.

Conclusion

In this article, you learned various ways to manage your content using the Ghost CMS. The Ghost blogging platform provides easy to use options to manage the content. Most options being self-explanatory, allows users to create posts, edit posts, publish and manage them by themselves, without having any need to additional training.
