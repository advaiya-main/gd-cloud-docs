Configure Themes and Settings for Ghost - Debian, Ubuntu

Difficulty: 2

Time: 30 minutes

Ghost is an open-source project for bloggers. It provides a simple blogging platform, is fast and easy to set up, and provides good customization options. This article helps you set up a Ghost blogging platform using Nginx on Debian or Ubuntu. The theme used is available as freeware from the Ghost Marketplace.

Prerequisites

  • Install and configure Ghost with NGINX - Ubuntu
  • Install and configure Ghost with NGINX - Debian

Change the Ghost Theme

  1. Log in and then move to the themes directory of Ghost.
    cd /var/www/ghost/content/themes/
  2. Install Git and git-core unless they are already installed.
    sudo apt-get update && sudo apt-get install git-core -y
  3. Clone the N-Coded theme (as an example) from the GitHub project.
    Sudo git clone https://github.com/polygonix/N-Coded.git
  4. Change the ownership of the Ghost user.
    sudo chown -R ghost:ghost N-Coded

    If you receive an invalid user error, please create a Ghost user and then retry.

Change the Ghost settings

  1. In a browser, type http://your_IP_address/ghost/settings/general. You will be redirected to the time setting page.
  2. Fill the required information about your blog and credentials on the setup form.
  3. Click OK, LET'S DO THIS.
  4. In the general settings page, select the theme from the Theme dropdown list, and click Save.
  5. Refresh your Ghost public home page to see the blog with the updated theme.

Conclusion

In this article, we reviewed the steps to configure themes and settings for Ghost on Debian or Ubuntu.
