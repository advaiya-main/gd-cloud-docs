Describes how to install Apache, MySQL, PHP (LAMP) stack on Ubuntu and Debian Linux.

Build a LAMP stack for Python - Ubuntu, Debian Linux

Difficulty: 3
Time: 4.00

The LAMP stack is a group of open source software used to get web servers up and running. The acronym stands for Linux, Apache, MySQL, and PHP. Ubuntu and Debian Linux uses the apt-get installer, which can update the system and install all of the required packages for each part of the LAMP stack. The steps in this article require the user to have root privileges.

Step One - Install Apache

Before installing any of the LAMP programs, we should update the package manager:

sudo apt-get update

Once the update is complete, we can install Apache:

sudo apt-get install apache2

Please make sure Apache is installed, bring up a Internet browser to your server's IP address (i.e. http://12.34.56.789). You should get the screen shot below if everything went good.

Install MySQL

  1. Run the apt-get update to ensure your server is up-to-date on all of its apt packages.
    sudo apt-get update
  2. Install MySQL.
    sudo apt-get install mysql-server php5-mysql
  3. Activate MySQL.
    sudo mysql_install_db
    MySQL will ask you for a db admin password. Skip this step for now.
  4. Secure your database server.
    sudo /usr/bin/mysql_secure_installation
The prompt will request the answer to a number of questions.
The following table contains the remainder of the questions you will be asked, the recommended response and some remarks regarding the consequences of not following the recommendation.

Question Suggested Answer Why / Why not?
Enter current password for root: Press Enter for none. Do not enter a password. If you do so, MySQL will not have sufficient access to system resources to function as a database server.
Do you want to change the root password?
y
 A password is needed to secure MySQL. Enter a strong password and write it down.
Remove anonymous users?
y
 The anonymous user allows anyone to login without requiring an account created by an admin. A hacker attempting to compromise your database server may try logging in as this user.
Disallow root login remotely?
y
 It is more secure to create a non-root user with admin permissions. Answer "no," if you need more flexibility to manage your db server.
Remove test database and access to it?
y
 If you have a reason to keep a test database, answer "no."
Reload privilege tables now?
y
 Why wait?

Step Three - Install PHP

PHP is an open source web scripting language that is widely used to build dynamic webpages.

To install PHP, in the same terminal type in these commands:

sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt

PHP should also be added to the apache configuration file:

sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt

Paste in this text block within the config.
